# Event-Manager-System
A Java Swing desktop application that allows users to manage online and physical events.  

## Features

- Add events
- Search events
- Save events to file
- Load events
- Java Swing GUI
- Object-Oriented Programming
- Inheritance
- Polymorphism
- File Handling

## Technologies

- Java
- Java Swing
- OOP
- File I/O

Included in repository.
